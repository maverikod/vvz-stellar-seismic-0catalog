# Задание для модели

## Автоматическое скачивание каталога астросейсмологических данных звёзд

### Цель

Скачать открытые каталоги астросейсмологических параметров звёзд из научных архивов, объединить их в единую таблицу и сохранить результат в архиве вместе с кодом, источниками и графиками.

Модель должна выполнить полный цикл:

1. скачать данные
2. объединить каталоги
3. очистить данные
4. построить базовые графики
5. сформировать архив результата

---

# Источники данных

Использовать открытые астрономические архивы.

### 1. MAST (NASA)

Архив миссий:

* Kepler
* TESS

Сайт:

[https://mast.stsci.edu](https://mast.stsci.edu)

Использовать API или библиотеку astroquery.

---

### 2. VizieR (CDS)

Каталоги астросейсмологии.

Использовать каталог Kepler Asteroseismic Catalog и аналогичные каталоги TESS.

Сайт:

[https://vizier.cds.unistra.fr](https://vizier.cds.unistra.fr)

---

### 3. Gaia DR3

Использовать для получения уточнённых значений:

* радиуса
* светимости
* расстояния

Сайт:

[https://gea.esac.esa.int/archive/](https://gea.esac.esa.int/archive/)

---

# Параметры, которые необходимо собрать

Для каждой звезды собрать следующие параметры.

### Основные параметры

star_id
catalog
ra
dec
mass
radius
luminosity
Teff
metallicity
distance

---

### Параметры колебаний

nu_max
delta_nu
mode_width
mode_amplitude

---

# Объём выборки

Скачать данные **не менее чем для 500 звёзд**.

Если возможно, увеличить выборку до **1000 и более объектов**.

---

# Очистка данных

Удалить записи:

* без значения radius
* без значения nu_max
* с ошибками измерений более 20%

---

# Формируемые таблицы

Создать две таблицы.

### stars_raw.csv

сырые данные, полученные из каталогов

---

### stars_clean.csv

очищенные данные со следующими колонками:

star_id
radius
mass
luminosity
nu_max
delta_nu
mode_width
Teff
distance

---

# Графики

Построить и сохранить три графика.

1. radius vs luminosity
2. nu_max vs radius
3. delta_nu vs mass

Сохранить графики в формате PNG.

---

# Требования к коду

Использовать Python 3.

Разрешённые библиотеки:

numpy
pandas
astroquery
astropy
matplotlib

Код должен быть разделён на два файла.

download_catalogs.py — скачивание данных
process_catalogs.py — обработка и очистка данных
